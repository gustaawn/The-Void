# The-Void
An experimental digital portfolio combining surrealism, minimalism and interactive design.
Focused on atmosphere, motion and emotional experience rather than conventional structure.

*English*
# Overview

This project explores the intersection between art and code.
The website functions as a living canvas where animation and interaction coexist.

The goal is not only to present work, but to create presence.

# Features

Minimalist surreal interface

Glassmorphism layout

Interactive “The Void” section

Custom music player with floating mini-player

Scroll-based animations using GSAP

Parallax effects

Custom cursor

Bilingual system (EN / PT-BR)

Fully responsive

# Technologies

HTML5

Tailwind CSS

JavaScript (Vanilla)

GSAP 3 and ScrollTrigger

Google Fonts
